# Replication Notes

This folder contains a short replication helper for the project.

The detailed workflow is described in:

docs/workflow_instructions.md

The analysis scripts are stored in:

src/

The main output tables and figures are stored in:

outputs/tables/
outputs/figures/

The complete workflow can be run with:

bash replication/run_all.sh

A valid Alpha Vantage API key is required for the news data collection step.
