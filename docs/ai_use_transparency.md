# AI Use Transparency

This document explains how AI tools were used during the project.

We used ChatGPT as a support tool while working on the project. It was mainly used to discuss possible research directions, understand coding steps, draft initial script structures, debug errors, and improve the organization of the documentation.

AI was also used to help explain preliminary results in simpler language. However, the group reviewed the generated suggestions before using them.

All scripts were run locally, and the outputs were checked by the group. The datasets, tables, figures, and regression results were inspected before being included in the project.

The final research framing, interpretation of results, writing decisions, and submission remain the responsibility of the group.

AI-generated suggestions were not used as final content without review.
